Movie Deck

Max Lewis

Application Overview: Allows users to log in, rate movies, and see other people's ratings.

Technology Overview: I plan on using Vue.js as the front end framework, and routing. I am also thinking of using either Tailwind or Bootstrap for easier styling. Firebase for authentication, and firestore for the database.

Last Updated: 11/4/2022

List of Features: Sign in, post rating, edit rating, see other people's ratings.

Future Feature List: Implement ratings from other websites, such as IMDb and Rotten Tomatoes. Implement a comment feature on people's reviews to have a conversation about their rating. Implement a feature so people can only rate real movies, and not allow them to pick a title of a movie that may not exist. Implement a feature for people to follow their favorite reviewers and only be able to see the reviews of the people they follow.

Basic Technical Features: Creating a UI using HTML, CSS. JS. Implement a database.

Advanced Technical Features: I will be trying to use Vue.js, which is something we have not learned in class. I will also be implementing authentication, which we also have not used in class.
